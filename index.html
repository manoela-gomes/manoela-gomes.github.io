Funções Matemáticas e de Arredondmento de Números

Função Uso Parâmetros Saída Exemplo
abs(x) calcula módulo de x
choose(x,y) calcula combinação de x em y
exp(x) exponencial de x
factorial(x) calcula fatorial de x
is.infinite(a/b) calcula se a/b é infinito
cos(x) calcula cosseno de x radianos
sin(x) calcula seno de x radianos
log(x,10) ou log10(x) calcula log de x na base 10
round(x,n) arredonda x com aproximação de n casas decimais
trunc(x) arredonda x em zero

median(x) calcula mediana dos valores de x
range(x) calcula amplitude dos valores de x

summary(x) retorna resumo de estatísticas descritivas

Funções de Estatística Inferencial

Função Uso Parâmetros Saída Exemplo
cor(var1,var2, method=método) correlação entre var1 e var2 calculada por método(Spearman,Kendall ou Pearson)
t.test(amostra1, amostra1, opções) realiza o teste t-Student para uma ou duas amostras utlizando opções(alternative,mu,paired,var.equal,conf.level) amostra1,amostra1 e opções //para uma amostra independente
amostra1 = c(14.9,13.4,14.5,13.5,15.0,13.9,14.9,16.4,14.6,15.4)
t.test(amostra1,mu=15)
//para duas amostras independentes
amostra1 = c(16.6,13.4,14.6,15.1,12.9,15.2,14.0,16.6,15.4,13.0) amostra2 = c(15.8,17.9,18.2,20.2,18.1,17.8,18.3,18.6,17.0,18.4) t.test(amostra1, amostra2, var.equal = TRUE)
//para duas amostras pareadas
antes = c(16.6,13.4,14.6,15.1,12.9,15.2,14.0,16.6,15.4,13.0)
depois = c(15.8,17.9,18.2,20.2,18.1,17.8,18.3,18.6,17.0,18.4)
t.test(antes,depois,paired=TRUE)
prop.test(x, n, p, opções) realiza o teste de proporções para uma ou duas amostras utilizando opções(correct,alternative,conf.level) //para uma proporção
prop.test(104,200,0.6,correct=F)
//para duas proporções
prop.test(c(104,50),c(200,95),correct=F)

Funções de Strings e Caracteres

Função Uso Parâmetros Saída Exemplo

paste(x, ly=" ", sep=" " ) insere "ly" no final de cada elemento de x separando por "sep"
substr(x,i,j) extrai do caractere i ao j da string x
nchar(x) número de caracteres de x
grep("a",x) quais elementos de x contém "a"
grep("a|b",x) quais elementos de x contém "a" ou "b"
grepl("abc",x)
strsplit(string,caractere) separa string em itens retirando caractere
startsWith(x,y)/endsWith(x,y) retorna se x começa com y/ retorna se x termina com y
cat(t1,t2) concatena t1 e t2
